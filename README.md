# 📝 Django Blog Project

A simple and elegant blog application built with **Django**, **HTML templates**, and styled using **TailwindCSS**. This project includes user authentication, blog post management, and a responsive UI.

## 🚀 Features

- User registration, login, and logout
- Create, edit, and delete blog posts
- Responsive UI with TailwindCSS
- Clean and minimal HTML templates
- Pagination and post previews
- Markdown support (optional)

## 🛠️ Tech Stack

- Python 3.x
- Django 4.x
- TailwindCSS (via Django-Tailwind or CDN)
- SQLite (default, can switch to PostgreSQL/MySQL)
